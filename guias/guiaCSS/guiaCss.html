Selecionar Tags do HTML para aplicar estilos

Estrutura

Essa é a estrutura para selecionar a quem será aplicado o estilo. 

selecionados { 
    estilo;
    estilo;
    estilo; 
}

Em que: selecionados são os elementos selecionados (separados por , ) e estilos é o estilo a ser aplicado nos selecionados (separados por ; ).

Seletores

Estes são quem usamos para identificar os elementos selecionados e seus respectivos metodos de utilização.

  • tag
    'nome da tag'{ ... }
  • class
    .'nome da classe'{ ... }
  • id
    #'id'{ ... }

Css interno ao HTML

Podemos aplicar um CSS diretamente no arquivo HTML, para isso utilizando a tag <style> CSS </style> diretamente dentro cabeçalho.
Em que: CSS é o codigo em CSS

Cor de fundo

A cor de fundo de um elemento é alterada pelo atributo background-color: 'cor';.
Em que: cor é o codigo da cor.

Cor da fonte

A cor da fonte(texto) de um elemento é alterada pelo atributo color: 'cor';.
Em que: cor é o codigo da cor.
