Este é um guia de funcionalidades do HTML

Estrutura Basica do html

Essa é a estrutura basica de um documnto HTML. 

<!DOCTYPE html>
<html lang="pt-br">
<head>                                            
    <meta charset="UTF-8">
    <title>Estrutura Basica do html</title>
</head>
<body>

</body>
</html>

Títulos

Usamos as tags <h1>...</h1>.
Em que: h1 é o titulo mais importante e h6 é o de menos importancia.

Paragrafos

Usamos a tag <p>...</p> para representar paragrafos em HTML.

Imagens

Usamos a tag <img src="x" alt="y"> para adicionar imagens no corpo do nosso html.
Em que: x é o caminho e y é o texto a ser exibido caso a imagem não carregue. ;

Quebra de linha

Podemos aplicar uma quebra de linha em HTML usando a tag <br>.

Comentarios

Podemos aplicar um comentario (texto que será ignorado pela maquina) em HTML usando a tag <!-- 'comentario' -->.
Em que: 'comentario' é o seu comentario.

Emojis

Podemos adicionar um emoji em HTML usando a estrutura&#x'codigo do emoji';.
Em que: 'codigo do emoji' é o codigo do seu emoji e pode ser obtido atravez de uma tabela que pode ser encontrada na internet.

Icone e Titulo da pagina

O icone da pagina é definido no cabeção pela tag <link rel="shortcut icon" href="favicon.ico" type="image/x-icon">.
Em que: 'favicon.ico' é o caminho do seu icone.

O titulo da pagina tambem é definido no cabeção pela tag <title>'titulo'</title>.
Em que: 'titulo' é o titulo da sua pagina.

Css interno ao HTML

Podemos aplicar um CSS diretamente no arquivo HTML, para isso utilizando a tag <style> CSS </style> diretamente dentro cabeçalho.
Em que: CSS é o codigo em CSS

Negrito, Italico, big, small,

Deixamos um texto em negrito com a tag <strong> texto </strong>.
Em que: texto é o texto que ficará em negrito.
